# Contributing to Kaleidoscope

Kaleidoscope is currently a single-author project. External contributions, including pull requests, are not accepted.

The repository is public so the design can be observed and read. Star or watch the repository to be notified when contribution opens.

When contribution opens, it will be on a **DCO sign-off only, no CLA** model. The rationale is documented in [section A.2 of the implementation roadmap](docs/roadmap/kaleidoscope-foss-implementation-roadmap.md). This file will be replaced with the full guidelines at that point.

— Andrea Laforgia, sole maintainer
