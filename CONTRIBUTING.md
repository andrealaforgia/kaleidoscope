# Contributing to Kaleidoscope

Kaleidoscope is currently a single-author project. External contributions, including pull requests, are not yet accepted.

The repository is public so the design can be observed and read. Star or watch the repository to be notified when contribution opens.

When contribution opens, the model will be simple. Kaleidoscope is dedicated to the public domain under [CC0-1.0](LICENSE), so contributions are themselves dedicated to the public domain on submission. There is no CLA. There is no DCO requirement. By submitting work to the repository you are simply releasing it to the public domain alongside the rest.

— Andrea Laforgia, sole maintainer
