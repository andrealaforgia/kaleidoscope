# Measurement scratch — speed-up-local-precommit-v0 (DESIGN)

Captured as-I-go so a mid-run drop does not lose the numbers.

## Structural facts (filesystem-measured, this session)

- Workspace crates: **26** (`Cargo.toml` members).
- Integration test binaries (`crates/**/tests/*.rs`): **165** — each is a
  separate `--all-targets` compile + run unit. `--lib` runs **none** of these
  by definition (it builds and runs only the in-crate `#[cfg(test)]` unit
  tests in `src/`, no `tests/*.rs` harness binary, no doctests).
- Of the 165, **26** match the durability/fsync family
  (`*wal_durability*`, `*snapshot*`, `*torn_tail*`, `*crash_durability*`,
  `*fsync_probe*`, `*snapshot_atomicity*`, `*filebacked_durable_recovery*`):
  cinder x4, pulse x6, lumen x4, ray x4, strata x3, sluice x3, beacon x1,
  log-query-api x1 (torn_tail). These are the per-record `sync_all` I/O-bound
  suites (ADR-0049/0060) — the root of the 10-20 min.
- Subprocess suites also excluded by `--lib`: aperture
  `slice_10_ingest_auth`, `serve_loop_error_surfacing`, `cli_smoke`,
  `probe_gold_runner`, `probe_refusal_visibility`; kaleidoscope-cli
  `*_roundtrip` / `*_smoke` / subcommand bins; beacon-server smoke/reload.

## Wall-clock — NOT captured this session (honest gap)

The DESIGN agent (Morgan) in this harness has **no shell-execution tool**
(only Read/Write/Edit/Glob/Grep). `cargo` / `time` could not be run from
DESIGN. The wall-clock seconds for fmt / clippy / deny / `--lib` and the
fast-hook total were therefore NOT measured here. Fabricating seconds would
violate the test-don't-assume / Earned-Trust principle, so they are left as
an explicit DELIVER measurement obligation on Apex (platform-architect), who
edits the hook and runs it for real.

## Decision basis

The cut is decided on the **structural** fact, which is the honest and
deterministic one: `--lib` excludes all 165 integration binaries incl. all
26 fsync-bound durability bins and every subprocess bin. The 10-20 min is
dominated by exactly those excluded bins. The residual fast hook is:
toolchain check + fmt + clippy(compile-bound) + deny + `--lib`(unit, no
fsync, no subprocess) — none of which carry the per-record `sync_all` cost.
The <= 5 min bar is a DELIVER-confirmed measurement (Apex), gated by the
US-01 timing AC, not a DESIGN-asserted number.
